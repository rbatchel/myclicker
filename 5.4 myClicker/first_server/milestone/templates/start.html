About myClicker

This web application provides a simple interface that allows professors to get feedback from students during lecture. Professors can both ask questions to students and view student responses in an organized manner, while students have the option to ask the professor questions about lecture material in real time and vote on the most relevant questions. A user can be enrolled in multiple courses at once and can either be a student or professor depending on the course.
